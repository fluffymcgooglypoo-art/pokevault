Database Setup

PokeVault needs a PostgreSQL database to store your card inventory and price history.

Paste the DATABASE_URL from your database provider.

Where to find your URL:
• Replit — open your project → Secrets → copy DATABASE_URL
• Supabase — Project Settings → Database → Connection String
• Neon — Dashboard → Connection Details
• Local — e.g. postgresql://postgres:password@localhost:5432/pokevault
